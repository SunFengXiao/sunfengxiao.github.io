# 孙风潇个人主页

现代化学术个人主页，展示研究成果、学术信息和联系方式。

## 目录结构

```

├── publications.html       # 论文发表 - 发表论文列表与统计
├── contact.html            # 联系我们 - 联系方式与学术链接
├── navbar.js               # 导航栏脚本 - 响应式菜单功能
```

## 页面说明

### 首页 (index.html)
- 展示个人照片、基本信息和学术简介
- 包含导航栏，快速访问其他页面
- 显示论文、引用数和 h-index 的实时统计

### 研究领域 (research.html)
- 详细介绍研究方向和学术成果
- 包括量子光学、量子信息、非高斯量子态等内容
- 展示代表性研究项目和理论成果

### 论文发表 (publications.html)
- 按年份组织已发表的 SCI 论文列表
- 每篇论文包含标题、作者、期刊、链接等信息
- 展示论文总数、引用次数和 h-index 统计

### 联系我们 (contact.html)
- 提供办公地点和电子邮箱
- 显示学术链接（Google Scholar、Web of Science、ResearchGate、ORCID）
- 学生招募信息

### 样式表 (style.css)
- 统一的颜色主题和排版设置
- 响应式布局，适配各种屏幕尺寸
- 卡片式设计、阴影效果等现代 UI 元素

### 导航脚本 (navbar.js)
- 响应式导航菜单功能
- 移动设备上的汉堡菜单支持
- 页面切换时的活动状态指示
